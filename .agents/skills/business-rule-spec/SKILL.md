# 业务规则规格模板

## 操作前置条件 ⚠️ 必填

| 操作 | 前置条件 | 不满足时处理 | 提示文案 |
|------|----------|--------------|----------|
| 删除订单 | 订单状态为草稿 | 按钮禁用 | "只能删除草稿状态的订单" |
| 提交审批 | 必填字段已填写 | 高亮未填字段 | "请填写必填项" |
| 发货 | 订单已支付 | 按钮隐藏 | - |
| 退款 | 订单已支付且未发货 | 按钮禁用 | "已发货订单不支持退款" |
| 编辑 | 当前用户有编辑权限 | 按钮隐藏 | - |
| 批量删除 | 选中项状态一致 | 按钮禁用 | "不能同时删除不同状态的数据" |

## 操作后置动作 ⚠️ 必填

| 操作 | 后置动作 | 执行时机 | 失败处理 |
|------|----------|----------|----------|
| 创建订单 | 发送通知给相关人 | 创建成功后 | 静默失败，不影响主流程 |
| 审批通过 | 更新订单状态+发通知 | 审批成功后 | 事务回滚 |
| 删除用户 | 清理关联数据 | 删除成功后 | 事务回滚 |
| 修改密码 | 清除所有登录态 | 修改成功后 | 静默失败 |
| 上传文件 | 生成缩略图 | 上传成功后 | 异步重试 |

## 批量操作规则 ⚠️ 必填

| 操作 | 最大数量 | 状态限制 | 部分失败处理 |
|------|----------|----------|--------------|
| 批量删除 | 100条 | 同一状态 | 显示失败项，成功项已删除 |
| 批量导出 | 1000条 | 无 | 超出提示分批导出 |
| 批量审批 | 50条 | 待审批状态 | 显示失败原因 |
| 批量修改 | 100条 | 无 | 事务回滚，全部失败 |

### 批量操作反馈
```
成功：toast "已成功处理N条数据"
部分成功：弹窗显示 "成功N条，失败M条" + 失败原因列表
全部失败：toast "操作失败：[原因]"
```

## 数据导入规则 ⚠️ 必填

| 属性 | 规则 |
|------|------|
| 文件格式 | xlsx/csv |
| 文件大小 | 最大10MB |
| 数据量 | 最大5000条 |
| 模板 | 提供下载模板 |
| 校验 | 先校验后导入 |
| 重复处理 | 跳过/覆盖/报错（可选） |
| 进度显示 | 显示导入进度 |
| 结果反馈 | 成功N条，失败M条，下载失败明细 |

## 数据导出规则 ⚠️ 必填

| 属性 | 规则 |
|------|------|
| 文件格式 | xlsx/csv |
| 数据量 | 最大10000条 |
| 超出处理 | 提示分批导出或后台异步导出 |
| 字段选择 | 支持自定义导出字段 |
| 敏感数据 | 脱敏处理（手机号、身份证） |
| 进度显示 | 大数据量显示进度 |
| 文件命名 | {功能名}_{日期}_{时间}.xlsx |

## 定时任务影响 ⚠️ 必填

| 定时任务 | 执行时间 | 影响数据 | 前端处理 |
|----------|----------|----------|----------|
| 订单超时取消 | 每5分钟 | 待支付订单 | 列表自动刷新 |
| 数据统计 | 每天凌晨 | 统计报表 | 显示数据更新时间 |
| 缓存清理 | 每小时 | 缓存数据 | 无需处理 |
| 消息推送 | 实时 | 消息列表 | WebSocket接收 |

## 数据有效期 ⚠️ 必填

| 数据类型 | 有效期 | 过期处理 | 用户提示 |
|----------|--------|----------|----------|
| 验证码 | 5分钟 | 重新获取 | "验证码已过期" |
| 支付链接 | 30分钟 | 重新生成 | "支付链接已过期" |
| 下载链接 | 24小时 | 重新生成 | "下载链接已过期" |
| 邀请链接 | 7天 | 重新邀请 | "邀请链接已过期" |
| 草稿 | 30天 | 自动删除 | 提前3天提醒 |

## 敏感操作确认 ⚠️ 必填

| 操作 | 确认方式 | 确认内容 |
|------|----------|----------|
| 删除数据 | 弹窗确认 | "确认删除？删除后不可恢复" |
| 批量删除 | 弹窗确认 | "确认删除选中的N条数据？" |
| 修改密码 | 输入原密码 | 验证原密码 |
| 注销账号 | 输入密码+验证码 | 二次验证 |
| 大额转账 | 短信验证码 | 二次验证 |
| 权限变更 | 弹窗确认 | "确认修改权限？" |
