# 页面规格模板

每个页面必须包含以下内容，评审前必须完整填写。

## 页面规格：[页面名称]

### 基本信息
| 属性 | 值 |
|------|-----|
| 页面名称 | |
| 页面路径 | /module/page |
| 所属模块 | |
| 关联需求 | Req-XXX |
| 页面类型 | 列表页/详情页/表单页/仪表盘 |

### 状态覆盖 ⚠️ 必填

| 状态 | 是否需要 | 设计稿链接 | 备注 |
|------|----------|------------|------|
| 正常态（有数据） | ✅ | | |
| 空态（无数据） | ✅/❌ | | 空态引导文案 |
| 加载态 | ✅/❌ | | 骨架屏/spinner |
| 错误态 | ✅/❌ | | 错误提示+重试 |
| 部分加载态 | ✅/❌ | | 列表加载更多 |
| 禁用态 | ✅/❌ | | 无权限时 |
| 编辑态 | ✅/❌ | | 表单编辑中 |
| 提交中态 | ✅/❌ | | 按钮loading |

### 交互状态 ⚠️ 必填

| 元素 | hover | focus | active | disabled | selected |
|------|-------|-------|--------|----------|----------|
| 主按钮 | ✅ | ✅ | ✅ | ✅ | - |
| 次按钮 | ✅ | ✅ | ✅ | ✅ | - |
| 输入框 | - | ✅ | - | ✅ | - |
| 下拉框 | ✅ | ✅ | - | ✅ | ✅ |
| 表格行 | ✅ | - | ✅ | - | ✅ |

### 边界场景 ⚠️ 必填

| 场景 | 处理方式 | 具体规则 |
|------|----------|----------|
| 文本超长 | 截断+省略号 | 最多显示N字符，hover显示tooltip |
| 数据超多 | 分页 | 每页10/20/50条，默认10 |
| 数据为空 | 空态组件 | 显示图标+文案+操作按钮 |
| 网络超时 | 错误提示 | "网络异常，请重试"+重试按钮 |
| 权限不足 | 隐藏/禁用 | 无权限时不显示或禁用 |

### 响应式设计（如需支持）

| 断点 | 屏幕宽度 | 布局变化 |
|------|----------|----------|
| 大屏 | ≥1440px | 默认布局 |
| 桌面 | 1200-1439px | 侧边栏收窄 |
| 平板 | 768-1199px | 侧边栏折叠 |
| 手机 | <768px | 单列布局 |

### 字段规格 ⚠️ 必填

| 字段名 | 显示名 | 类型 | 格式 | 校验规则 | 错误提示 | 必填 |
|--------|--------|------|------|----------|----------|------|
| name | 名称 | string | - | 2-50字符 | "名称长度2-50字符" | ✅ |
| status | 状态 | enum | 标签 | - | - | ✅ |
| createTime | 创建时间 | datetime | YYYY-MM-DD HH:mm | - | - | - |
| amount | 金额 | number | ¥0.00 | ≥0 | "金额格式不正确" | ✅ |

### 操作反馈 ⚠️ 必填

| 操作 | 成功反馈 | 失败反馈 | 需要确认 | 确认文案 |
|------|----------|----------|----------|----------|
| 新建 | toast "创建成功" | toast 错误信息 | ❌ | - |
| 删除 | toast "删除成功" | toast 错误信息 | ✅ | "确认删除？" |
| 提交 | toast "提交成功" | toast 错误信息 | ✅ | "确认提交？" |

### 权限视图 ⚠️ 必填

| 角色 | 可见元素 | 可操作 | 差异说明 |
|------|----------|--------|----------|
| 管理员 | 全部 | 全部 | - |
| 普通用户 | 隐藏删除按钮 | 只能编辑自己的 | |
| 游客 | 公开数据 | 无 | 显示登录引导 |

### 页面间跳转

| 触发元素 | 目标页面 | 跳转方式 | 携带参数 |
|----------|----------|----------|----------|
| 列表行点击 | 详情页 | 当前页跳转 | id |
| 新建按钮 | 新建页 | 当前页跳转 | - |
| 返回按钮 | 列表页 | 返回上一页 | - |
