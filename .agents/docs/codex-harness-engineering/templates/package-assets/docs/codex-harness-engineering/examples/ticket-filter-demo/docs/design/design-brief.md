# Design Brief

## 1. 设计目标

- 功能主题: 工单后台筛选体验
- 要解决的问题: 信息密度高但不能压迫，筛选操作要快，批量管理感要强

## 2. 视觉方向

- 主题风格: 企业后台、克制、专业
- 品牌气质: 高密度但清晰、可信赖
- 避免: 营销 landing page 风格、过多插画、过强渐变

## 3. 色彩策略

- 主色: 蓝灰系
- 强调色: 橙色用于高优先级和 CTA
- 错误色: 红色

## 4. 布局与组件原则

- 页面布局: 顶部筛选条 + 主数据表
- 筛选条应支持标签化回显
- 保存视图使用小型弹窗，不要切走主页面

## 5. 交互原则

- 主按钮: 保存视图
- 次按钮: 清空筛选
- 加载态: 表格骨架屏 + 按钮禁用
- 错误态: 内联错误提示 + 重试
