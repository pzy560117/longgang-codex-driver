## 阈值

| 范围 | 最低要求 | 备注 |
| --- | --- | --- |
| 业务规则 | 80% | 定价、库存、权限、校验 |
| API / 服务层 | 80% | 包含成功和错误分支 |
| UI 组件 | 覆盖关键状态 | 相比原始行覆盖率，更优先覆盖交互 / 状态 |
| E2E | 覆盖 P0/P1 主流程 | 不按行覆盖率衡量 |
| 视觉 | P0 UI + 高风险状态 | 必须提供截图证据 |
